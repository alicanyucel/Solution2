## 🎯 Hedef
LinkedIn'de 5 dakikada otomatik iş başvurusu yapmaya başlayın!

---

## 📋 Gereksinimler
- ✅ Google Chrome tarayıcı
- ✅ LinkedIn hesabı
- ✅ 5 dakika zamanınız

---

## 🚀 3 Adımda Kurulum

### 1. Icon Dosyalarını Oluşturun (2 dakika)

```bash
1. LinkedInJobExtension/icons/icon-generator.html dosyasını Chrome'da açın
2. Her üç butona tıklayarak iconları indirin:
   - 📥 icon16.png İndir
   - 📥 icon48.png İndir  
   - 📥 icon128.png İndir
3. İndirilen dosyaları LinkedInJobExtension/icons/ klasörüne taşıyın
```

### 2. Extension'ı Yükleyin (2 dakika)

```bash
1. Chrome'da chrome://extensions/ adresine gidin
2. Sağ üstten "Developer mode" aktif edin
3. "Load unpacked" butonuna tıklayın
4. "LinkedInJobExtension" klasörünü seçin
5. ✅ Tamam!
```

### 3. İlk Başvuruyu Yapın (1 dakika)

```bash
1. LinkedIn'e giriş yapın
2. https://www.linkedin.com/jobs/ sayfasına gidin
3. Extension ikonuna tıklayın (araç çubuğunda)
4. Pozisyon girin: "Software Developer"
5. Lokasyon girin: "Turkey"
6. "Otomasyonu Başlat" butonuna bas!
```
